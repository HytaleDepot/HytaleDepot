# 🚀 Hytale Depot

🌐 https://hytaledepot.net

**Hytale Depot** is a community platform focused on the game **Hytale**.

The goal of the project is to create a **central hub for creators, developers, and server owners** where they can publish, discover, and share content.

The platform aims to support things like:

- 📦 **Mods & plugins**
- 🧱 **World builds and schematics**
- 🎨 **3D models and creative assets**
- 🧰 **Developer tools and resources**
- 🌐 **A marketplace ecosystem for creators**

The long-term vision is to build something similar to major modding platforms, but **dedicated entirely to the Hytale ecosystem**.
